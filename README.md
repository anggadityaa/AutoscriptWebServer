# AutoscriptWebServer
Ini adalah script auto install untuk web server


## CARA MENJALANKAN SCRIPT

```sh
git clone https://github.com/anggadityaa/AutoscriptWebServer.git
```
```sh
cd AutoscriptWebServer
```
```sh
chmod +x install.sh
```
```sh
sudo su
```
```sh
./install.sh
```
